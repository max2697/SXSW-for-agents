FAQ

Practical answers for using the website and data feeds.

What is this website?

A static SXSW 2026 schedule mirror designed for both humans and agents.

Where should agents start?

Start with /schedule.manifest.json, then use /agent-schedule.v1.json.

How do I track updates?

Read /changes.ndjson. It includes added, modified, removed, and cancelled records plus tombstones.

How fresh is the data?

Check freshness fields in /schedule.manifest.json: source_snapshot_at, expected_next_refresh_by, and data_staleness.

Can I browse by day?

Yes. Use /schedule/index.html and each date page for filterable tables.

Can I use this with Claude/ChatGPT/Gemini?

Yes. Use /prompts/index.html for copy/paste prompt examples and a prompt builder.
